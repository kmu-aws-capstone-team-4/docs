# UC-NAV-001 소개 페이지 메인 화면 접근

| 항목 | 내용 |
|------|------|
| **액터** | 비인증 사용자, 인증 사용자, 일반 사용자, Pro 사용자 |
| **목표** | 서비스 소개 페이지 메인 화면에 접근한다 |
| **사전 조건** | 없음 (로그인 불필요) |
| **사후 조건** | 소개 페이지가 표시된다 |

---

## 기본 흐름

1. 사용자가 서비스 루트 URL에 접근한다
2. 시스템이 로그인 여부를 확인한다
3. 비로그인 상태이면 소개 페이지 메인 화면을 표시한다
4. 화면에는 서비스 소개, 주요 기능, 요금제 안내, 회원가입/로그인 버튼이 포함된다

---

## 대안 흐름

- **3a.** 로그인 상태인 경우 개인화 메인 화면으로 이동한다 → [UC-NAV-002-로그인_후_개인화_메인_화면_접근.md](./UC-NAV-002-로그인_후_개인화_메인_화면_접근.md)

## 비고

- 이 화면은 인증 여부와 무관하게 항상 접근 가능하다
- 이메일 미인증 사용자도 이 화면에는 리다이렉트 없이 접근할 수 있다
