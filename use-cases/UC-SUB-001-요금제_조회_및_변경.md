# UC-SUB-001 요금제 조회 및 변경

| 항목 | 내용 |
|------|------|
| **액터** | 일반 사용자, Pro 사용자 |
| **목표** | 현재 요금제를 확인하고 업그레이드 또는 변경한다 |
| **사전 조건** | 로그인 상태 |
| **사후 조건** | 구독이 활성화되고 Pro 기능을 즉시 이용 가능하다 |

---

## 기본 흐름

1. 사용자가 요금제 페이지에 접근한다
2. 시스템이 무료/Pro 요금제 기능 비교표를 표시한다
3. 사용자가 Pro 요금제를 선택하고 결제를 진행한다
4. 시스템이 구독을 활성화하고 Pro 기능을 즉시 이용 가능하게 한다

---

## 비고

- 요금제별 기능 한도는 `feature_quotas` 테이블에서 관리된다
- 스트릭 보상으로 지급된 Pro 기능 사용 횟수는 구독 여부와 무관하게 사용 가능하다
