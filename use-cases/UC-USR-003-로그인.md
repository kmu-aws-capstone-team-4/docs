# UC-USR-003 로그인

| 항목 | 내용 |
|------|------|
| **액터** | 비인증 사용자 |
| **목표** | 서비스에 로그인한다 |
| **사전 조건** | 가입된 계정 보유, 이메일 인증 완료 |
| **사후 조건** | 세션/토큰이 발급되고 메인 페이지로 이동한다 |

---

## 기본 흐름

1. 사용자가 이메일과 비밀번호를 입력한다
2. 시스템이 자격증명을 검증한다
3. 시스템이 세션/토큰을 발급하고 메인 페이지로 이동한다

---

## 대안 흐름

- **2a.** 이메일 또는 비밀번호가 틀린 경우 오류 메시지를 표시한다
- **2b.** 이메일 인증이 완료되지 않은 경우 인증 링크 재전송 옵션을 제공한다
