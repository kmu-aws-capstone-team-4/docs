# 유즈케이스 목록 (Use Cases)

> 가상면접 웹 서비스의 전체 유즈케이스 인덱스

---

## 파일 명명 규칙

```
UC-{카테고리}-{넘버링}-{요약_제목}.md
```

| 카테고리 코드 | 설명 |
|----------------------|------|
| `USR` (USER)         | 사용자 계정 (회원가입, 로그인, 프로필) |
| `NAV` (NAVIGATION)   | 화면 접근 및 네비게이션 (메인 화면, 소개 페이지) |
| `DOC` (DOCUMENT)     | 문서 업로드 (이력서, 채용공고) |
| `INT` (INTERVIEW)    | 가상 면접 (설정, 진행, 리포트) |
| `GMF` (GAMIFICATION) | 게이미피케이션 (스트릭, 보상) |
| `SUB` (SUBCRIBE)     | 요금제 및 구독 |
| `PVC` (PRIVACY)      | 개인정보 및 동의 |

---

## 액터 (Actors)

| 액터 | 영어명 | 설명 |
|------|--------|------|
| **미가입 방문자** | `Guest` | 회원가입 또는 로그인 전 상태의 방문자 |
| **이메일 미인증 사용자** | `UnconfirmedUser` | 회원가입은 완료했으나 이메일 인증 링크를 클릭하지 않은 상태의 사용자 (`is_email_verified = false`) |
| **프로필 미작성 사용자** | `OnboardingUser` | 이메일 인증은 완료했으나 프로필(닉네임, 직군 등)을 아직 작성하지 않은 상태의 사용자 (`is_profile_complete = false`) |
| **인증 사용자** | `AuthenticatedUser` | 이메일 인증 및 프로필 작성까지 모두 완료한 사용자의 총칭. 일반 사용자와 Pro 사용자를 포함함 |
| **일반 사용자** | `FreeUser` | 인증 사용자 중 무료 요금제를 이용 중인 사용자 |
| **Pro 사용자** | `ProUser` | 인증 사용자 중 Pro 요금제를 구독 중인 사용자 |
| **시스템** | `System` | 백엔드 서버, LLM, 벡터 DB 등 자동화된 처리 주체 |
