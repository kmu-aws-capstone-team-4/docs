# UC-USR-002 프로필 최초 작성

| 항목 | 내용 |
|------|------|
| **액터** | 인증 사용자 |
| **목표** | 서비스 이용을 위한 기본 프로필을 작성한다 |
| **사전 조건** | 이메일 인증 완료 후 서비스 접근 |
| **사후 조건** | 프로필이 저장되어 면접 질문 생성에 반영 가능한 상태가 된다 |

---

## 기본 흐름

1. 사용자가 본명(이름)을 입력한다
2. 사용자가 희망 직군을 선택한다 (전체 사회 직군 대상)
3. 사용자가 현재 직업을 입력한다
4. 시스템이 프로필을 저장하고 서비스 메인으로 이동한다

---

## 대안 흐름

- **4a.** 사용자는 프로필 작성을 건너뛸 수 없다.

---

## 비고

- 희망 직군과 직업 정보는 이후 가상 면접 질문 생성 시 그때그때 동적으로 반영된다
- 프로필 정보는 [UC-USR-004-개인정보수정.md](./UC-USR-004-개인정보수정.md)를 통해 언제든지 변경 가능하다
