# UC-USR-001 회원가입

| 항목 | 내용 |
|------|------|
| **액터** | 비인증 사용자 |
| **목표** | 서비스 계정을 생성한다 |
| **사전 조건** | 유효한 이메일 주소 보유 |
| **사후 조건** | 계정이 생성되고 이메일 인증이 완료된다 (프로필 작성은 별도 진행) |

---

## 기본 흐름

1. 사용자가 이메일, 비밀번호를 입력한다.
2. 회원가입을 버튼을 눌러 회원가입을 요청한다
3. 시스템이 이메일 중복 여부를 확인한다
4. 시스템이 계정을 생성하고 인증 링크를 이메일로 전송한다
5. 사용자가 이메일의 인증 링크를 클릭한다
6. 시스템이 이메일 인증을 완료 처리한다
7. 시스템이 프로필 작성 페이지로 안내한다

---

## 대안 흐름

- **2a.** 이미 가입된 이메일인 경우 오류 메시지를 표시한다

- **4a.** 인증 링크가 만료될 경우에 대해서 이메일 인증 링크 발송 화면으로 이동하여 재전송 옵션을 제공한다 → [UC-USR-005-이메일_인증_링크_발송_화면_접근.md](./UC-USR-005-이메일_인증_링크_발송_화면_접근.md)
