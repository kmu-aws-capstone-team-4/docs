# UC-INT-002 꼬리질문 방식 면접 진행

| 항목 | 내용 |
|------|------|
| **액터** | 일반 사용자, Pro 사용자 |
| **목표** | 꼬리질문 방식으로 심층 면접을 진행한다 |
| **사전 조건** | 면접 세션 시작 완료 ([UC-INT-001](./UC-INT-001-면접설정및시작.md)), 꼬리질문 방식 선택 |
| **사후 조건** | 모든 질문이 완료되고 면접 세션이 종료된다 |

---

## 기본 흐름

1. 가상 면접관이 TTS 음성과 화면 텍스트로 첫 번째 질문을 제시한다
2. 답변 시작
   - **연습 모드**: 사용자가 준비 완료 버튼을 눌러 답변을 시작한다
   - **실전 모드**: 5~30초 랜덤 대기 후 자동으로 답변 녹화가 시작된다
3. 사용자가 음성으로 답변하며 STT가 실시간으로 텍스트를 변환한다
4. 시스템이 침묵(5초 이상, 특정 데시벨 이하)을 감지하고 횟수·맥락을 기록한다
5. 시선 추적이 활성화된 경우, 고개/시선 이탈 빈도를 실시간으로 측정·기록한다
6. 사용자가 답변을 완료한다
7. 시스템이 답변 내용, 이전 질문, 사용자 자료를 종합하여 꼬리 질문 1~3개를 생성한다
8. 2~7 과정을 반복한다
9. 모든 질문이 완료되면 면접 세션이 종료되고 리포트 생성으로 이어진다

---

## 대안 흐름

- **6a.** 사용자가 답변 도중 세션을 중단하면 `abandoned` 상태로 저장된다

---

## 비고

- 꼬리 질문은 최소 1개, 최대 3개 생성된다
- 가상 면접관은 SVG 기반 인간형 캐릭터로 화면 중앙에 표시되며, TTS 음성과 함께 질문을 전달한다
- TTS 오디오는 서버에서 생성하여 제공한다
