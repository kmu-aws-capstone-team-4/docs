# UC-INT-004 면접 리뷰 리포트 확인

| 항목 | 내용 |
|------|------|
| **액터** | 일반 사용자, Pro 사용자 |
| **목표** | 면접 세션에 대한 종합 피드백을 확인한다 |
| **사전 조건** | 면접 세션 종료 |
| **사후 조건** | 리뷰 리포트가 생성되어 사용자에게 제공된다 |

---

## 기본 흐름

1. 시스템이 면접 세션 데이터를 분석하여 리뷰 리포트를 생성한다
2. 사용자가 리포트를 확인한다
3. 리포트에는 다음 항목이 포함된다
   - 질문별 답변 내용 및 평가
   - 침묵 발생 횟수 및 맥락 분석
   - 시선 이탈 빈도 분석 (해당 기능 활성화 시)
   - 전반적인 면접 수행 피드백 및 개선 제안

---

## 비고

- 리포트 상세 수준은 요금제에 따라 다를 수 있다 (Pro 사용자는 더 상세한 분석 제공)
- 리포트는 이전 세션 목록에서 다시 조회할 수 있다
