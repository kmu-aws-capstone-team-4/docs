## 1. 경쟁 환경 분석

### 1.1 글로벌 주요 플레이어

**Final Round AI** — AI Interview Copilot이라는 실시간 면접 보조 도구로 급성장. 월 $96~300 과금, 1,000만+ 사용자 주장. 2025년 1월 오버서브스크라이브된 시드 라운드에서 $6.88M 확보. 실시간 면접 중 답변 제안을 제공하는 "치트 도구"적 성격이 윤리적 논란을 야기하고 있으며, 본 프로젝트와는 방향이 다르다(연습 vs 치팅).

**Yoodli** — AI 음성 코칭 플랫폼. 개인 월 $8, B2B 엔터프라이즈 중심. 2025년 12월 기준 기업가치 $300M+(6개월 만에 3배 상승), 900% 매출 성장, $60M 누적 투자. Google, Snowflake, Databricks 등 기업 고객 확보. 면접 연습보다 넓은 "커뮤니케이션 코칭" 포지셔닝.

**Interviewing.io** — 익명의 FAANG 엔지니어와 실제 모의 면접. 세션당 $179+. 총 $13M 투자. 사람 면접관 매칭이 핵심이라 AI 자동화와는 다른 시장.

**Big Interview** — 200만+ 사용자, 700+ 대학 파트너. 월 $39~299. 비디오 녹화 + AI 피드백. 대학 B2B 채널이 강력하여, B2B 비즈니스 모델의 검증 사례.

**Google Interview Warmup** — 완전 무료. 음성 인식으로 답변 녹음, 패턴 인사이트(반복 용어, 직무 관련 용어, 가장 많이 다룬 주제) 제공. 점수나 평가 없이 자기 성찰 유도. 무료라는 장점이 있지만 피드백이 제한적.

## 2. 게이미피케이션: 최고 우선순위 리텐션 전략

### 2.1 Duolingo의 검증된 메커니즘 적용

교육 앱의 리텐션은 최악의 카테고리에 속한다: Day 1 ~14-15%, Day 30 ~2-3%. Duolingo의 스트릭 시스템은 가장 검증된 대응책이며, 면접 연습에 직접 적용 가능하다.

**스트릭 효과 데이터**:
- 7일 스트릭 달성 사용자는 코스 완료율이 **3.6배 높음**
- 스트릭이 홈 화면 위젯으로 노출될 때 커밋먼트 **60% 증가**
- Streak Freeze(하루 미스 시 스트릭 유지)로 이탈 위험 사용자의 이탈률 **21% 감소**
- 스트릭과 일일 목표 분리(질문 1개만 완료해도 스트릭 유지)로 Day 14 리텐션 **3.3% 증가**, DAU **1% 증가**

**XP·리더보드 효과**: XP 스타일 리더보드가 주간 레슨 완료율 **40% 향상**. 배지가 완료율 **30% 향상**.

### 2.2 RICE 스코어링 기반 기능 우선순위

| 기능 | Reach | Impact | Confidence | Effort | RICE Score |
|------|-------|--------|-----------|--------|-----------|
| **스트릭 시스템** | 10,000 | 3 | 0.9 | 1.2 | **22.8** |
| 일일 질문 배달 | 10,000 | 2.5 | 0.9 | 1.5 | 15.0 |
| AI 피드백 (텍스트) | 8,000 | 2.5 | 0.8 | 2.5 | 8.1 |
| 이력서 기반 개인화 | 5,000 | 2 | 0.7 | 2.0 | 3.5 |
| 회사별 트랙 | 3,000 | 2 | 0.8 | 1.5 | 3.2 |
| 비디오 녹화 (V2) | 2,000 | 3 | 0.6 | 3.0 | 1.2 |

**결론**: 스트릭 시스템이 RICE 최고 점수(22.8)로, MVP Day 1부터 구현해야 할 **단일 최고 임팩트 기능**이다.

### 2.3 매일메일에 맞춤화된 게이미피케이션 설계

**스트릭 체계**:
- 최소 완료 조건: 하루 1개 질문 답변 (부담감 최소화)
- 스트릭 연속 일수 시각적 표시 (불꽃 아이콘 🔥)
- Streak Freeze: 무료 월 1회, 프리미엄 월 2회
- 마일스톤 축하: 7일, 30일, 100일, 365일 배지
- 스트릭 위험 알림: 당일 미완료 시 저녁 알림톡/이메일 발송

**XP 시스템**:
- 질문 답변: +10 XP
- 피드백 확인: +5 XP
- 연속 3일: 보너스 +20 XP
- 고난도 질문 완료: 추가 +15 XP
- 주간 리더보드 (동일 역할/직무 그룹 내)

**간격 반복 레이어**:
- 낮은 점수를 받은 질문을 2, 3, 5, 7일 간격으로 재출제 ("2357 방법")
- 성과 기반 난이도 자동 조정
- LLM으로 약점 영역 질문의 변형 자동 생성

## 3. 알림 채널별 전략: 카카오톡 알림톡 + 이메일

### 3.1 채널별 역할 분담

| 기능 | 카카오톡 알림톡 (1차) | 이메일 - AWS SES (2차) |
|------|---------------------|---------------------|
| **오늘의 질문 알림** | ✅ 짧은 미리보기 + CTA 링크 | ✅ 상세 질문 + 힌트 + 참고 자료 |
| **평가 완료 알림** | ✅ 점수 요약 + 상세 보기 링크 | ✅ 전체 피드백 리포트 |
| **스트릭 위험 알림** | ✅ 저녁 푸시 "오늘 아직 답변 안 했어요!" | ❌ |
| **주간 리포트** | ❌ | ✅ 주간 성과 요약, 트렌드, 다음 주 추천 |
| **마일스톤/배지** | ✅ 축하 메시지 | ❌ |
| **계정/결제** | ❌ | ✅ 가입 확인, 결제 영수증, 비밀번호 재설정 |

### 3.2 최적 발송 시간

- **오전 질문 알림**: 07:00~09:00 KST (출근 시간대, 아침 루틴 형성)
- **스트릭 위험 알림**: 20:00~21:00 KST (마감 전 마지막 기회)
- **주간 리포트**: 월요일 08:00 KST (새 주의 시작과 함께)

### 3.3 개인화된 제목줄/템플릿

알림톡 템플릿 예시:
```
[매일메일] 🎯 오늘의 면접 질문 #47
"당신이 팀에서 갈등을 해결했던 경험을 STAR 기법으로 설명해주세요."

🔥 연속 {streak}일째 도전 중!
▶ 답변하러 가기: {url}
```

이메일 교육/훈련 부문 벤치마크: 오픈율 ~35.6%, CTR ~3%. 개인화된 제목줄은 오픈율을 50% 향상시킨다. 한국 사용자의 경우 카카오톡 알림톡이 이메일 오픈율(70-80% vs 35%) 대비 2배 이상 높은 도달률을 보인다.

## 4. 질문 개인화 전략

### 4.1 회사별 면접 구조 반영

면접 질문은 회사마다 극적으로 다르다:

**글로벌 기업**:
- Amazon: 16개 리더십 원칙 기반 STAR 행동 면접 중심
- Google: 코딩(LeetCode 중상), 시스템 디자인, "Googleyness" 문화 적합성
- Microsoft: 코딩 + 시스템 디자인 + 행동면접 균형형

**한국 기업**:
- 삼성: GSAT + 다단계 인성면접·직무면접
- 네이버/카카오: 스타트업형 기술면접, 코딩 테스트 + 시스템 디자인
- 쿠팡: FAANG식 HackerRank + 시스템 디자인
- LG/SK/현대: AI역량검사 + 인성면접 + 직무면접

### 4.2 질문 데이터베이스 설계

**초기 출시**: 500~1,000개 큐레이션된 질문, 다음 태그로 분류:
- 회사 (삼성, 네이버, 카카오, Amazon, Google 등)
- 역할 (Backend, Frontend, PM, Data Science 등)
- 난이도 (주니어/미드/시니어)
- 유형 (행동 30-40%, 기술 25-35%, 시스템 디자인 15-25%, 케이스 스터디 10-15%, 문화적합 5-10%)

**동적 질문 생성**: LLM이 이력서·자소서 기반으로 변형 질문을 자동 생성. 분기별 업데이트로 커뮤니티 기여와 Glassdoor/Blind/한국 커뮤니티 패턴 반영.

### 4.3 자소서→면접 파이프라인 (한국 고유 차별화)

한국 채용의 독특한 흐름에서 자소서(자기소개서) 내용이 면접 질문을 직접 형성한다. 이 파이프라인은 경쟁자가 쉽게 복제할 수 없는 문화적 깊이를 요구하며, 핵심 차별화 요소가 된다.

**구현 방식**:
1. 사용자가 자소서 업로드 (또는 텍스트 입력)
2. LLM이 자소서의 핵심 주장·경험·수치를 추출
3. 각 주장에 대해 "꼬리질문"(심화 질문) 자동 생성
   - "이 프로젝트에서 구체적으로 본인의 역할은 무엇이었나요?"
   - "30% 성능 향상이라고 하셨는데, 어떤 지표로 측정했나요?"
4. 모범 답변과 함께 제공

## 5. 피드백 메커니즘 설계

### 5.1 다층적 피드백 구조

단순 점수를 넘어, 각 평가는 다음을 포함해야 한다:

**콘텐츠 품질**: 관련성, 깊이, 핵심 역량 시연 여부
**구조 평가**: STAR/SOAR 기법 준수, 논리적 구성
- Action 컴포넌트가 답변의 60~70%를 차지해야 함
- Result는 정량적 수치(지표, 퍼센트, 금액)를 포함해야 함
**전달 방식 (V2)**: 발화 속도, 필러 워드, 톤 변화, 자신감
**비언어적 (V2)**: 시선 접촉, 바디랭귀지, 자세

### 5.2 행동 가능한 피드백 원칙

"답변을 개선하세요" ❌
"Action 섹션을 정량적 결과를 포함하도록 재구성하세요. 예: '배포 시간을 기존 30분에서 5분으로 83% 단축했습니다'" ✅

각 피드백에는:
- 구체적 개선 팁 (행동 가능한)
- 모범 답변과의 비교
- 사용자 답변과 모범 답변의 핵심 차이점 하이라이트

### 6.1 MVP (1~4개월, 텍스트 기반)

### 6.2 V2 (5~9개월, 영상/음성)

| 기능 | 상세 |
|------|------|
| 웹캠 영상 녹화 | 비동기 처리 (배치) |
| STT 변환 | CLOVA Speech + Whisper |
| 필러 워드 감지 | "음", "어", "그" + 영어 필러 |

## 7. 사용자 여정 설계

### 7.1 신규 사용자 온보딩 (첫 5분 "Aha Moment")

```
1. 카카오/네이버/이메일 간편 가입 (30초)
2. 역할 선택 (백엔드/프론트엔드/PM/데이터 등) (15초)
3. 목표 회사 선택 (삼성/네이버/카카오/기타) (15초)
4. 일일 질문 개수 설정 (1~5개) (10초)
5. 알림 시간 설정 (07:00/08:00/09:00) (10초)
6. 이력서 업로드 (선택, 나중에도 가능) (0~60초)
7. 첫 번째 질문 즉시 제공 → 답변 → AI 피드백 (3분)
   → "Aha Moment": "이런 구체적 피드백을 매일 받을 수 있다!"
```

### 7.2 일일 루프 (핵심 리텐션 사이클)

```
[알림 수신] → [질문 확인] → [답변 작성] → [AI 피드백 확인]
    ↑                                          ↓
    └── 스트릭 유지 + XP 획득 ← ← ← ← ← ← ← ┘
```

### 7.3 면접 준비 집중 모드 (V2)

면접일이 설정되면 D-day 기반 커리큘럼 자동 생성:
- D-14: 기본 역량 질문 + 자소서 분석
- D-7: 회사 특화 질문 집중
- D-3: 영상 녹화 모의 면접
- D-1: 최종 리뷰 + 핵심 포인트 정리

## 8. 차별화 요약: "왜 매일메일인가"

| 경쟁자 | 매일메일의 차별점 |
|--------|-----------------|
| Final Round AI | 치팅 도구가 아닌 실력 향상 도구. 윤리적 포지셔닝 |
| Yoodli | 한국어 네이티브 + 한국 면접 문화 특화 |
| 사람인 AI면접 | 독립 제품, 매일 습관 형성, 프리미엄 가격 |
| 제네시스랩 | B2C 중심, 자소서→면접 파이프라인 |
| Google Warmup | 상세 AI 평가 + 점수 + 개인화 |
| Big Interview | 한국어 지원 + 한국 기업 트랙 |

**핵심 인사이트**: AI 정교함이 아닌 **일일 습관 루프가 핵심 제품 차별화**다. Duolingo가 스트릭 메커니즘으로 교육 카테고리를 정의했듯, 면접 준비 영역에서 이 원리를 적용한 플랫폼은 아직 없다.

1. **한국 취업준비생의 일일 면접 연습 완료율은 어떻게 예측할 수 있을까?** Duolingo의 레슨 완료율 데이터와 한국 교육 앱 사용 패턴을 교차 분석하여 현실적인 DAU 목표를 설정해야 한다.
