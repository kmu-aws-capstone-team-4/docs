# UC-NAV-002 로그인 후 개인화 메인 화면 접근

| 항목 | 내용 |
|------|------|
| **액터** | 일반 사용자, Pro 사용자 |
| **목표** | 로그인한 사용자에게 개인화된 메인 화면을 제공한다 |
| **사전 조건** | 로그인 상태, 이메일 인증 완료, 프로필 작성 완료 |
| **사후 조건** | 개인화 메인 화면이 표시된다 |

---

## 기본 흐름

1. 사용자가 서비스 루트 URL 또는 메인 화면 경로에 접근한다
2. 시스템이 로그인 및 이메일 인증 여부를 확인한다
3. 시스템이 개인화 메인 화면을 구성하여 표시한다
4. 화면에는 다음 항목이 포함된다
   - 사용자 닉네임 및 현재 요금제 표시
   - 현재 스트릭 현황 및 오늘 면접 참여 여부
   - 가상 면접 시작 버튼
   - 최근 면접 세션 이력 및 리뷰 리포트 바로가기
   - 업로드된 이력서·채용공고 현황 요약

---

## 대안 흐름

- **2a.** 비로그인 상태이면 소개 페이지로 이동한다 → [UC-NAV-001-소개_페이지_메인_화면_접근.md](./UC-NAV-001-소개_페이지_메인_화면_접근.md)
- **2b.** 로그인 상태이나 이메일 미인증 상태이면 이메일 인증 링크 발송 화면으로 이동한다 → [UC-USR-005-이메일_인증_링크_발송_화면_접근.md](./UC-USR-005-이메일_인증_링크_발송_화면_접근.md)
- **2c.** 로그인 및 이메일 인증은 완료되었으나 프로필 미작성 상태이면 프로필 작성 화면으로 이동한다 → [UC-USR-002-프로필_최초_작성.md](./UC-USR-002-프로필_최초_작성.md)

---

## 비고

- 개인화 메인 화면은 사용자의 직군, 스트릭, 최근 면접 이력을 기반으로 구성된다
- Pro 사용자에게는 추가 기능 접근 버튼(시선 추적, 실전 모드 등)이 활성화된 상태로 표시된다
