# UC-PVC-001 데이터 수집 동의 처리

| 항목 | 내용 |
|------|------|
| **액터** | 비인증 사용자 (회원가입 시), 일반 사용자 |
| **목표** | 영상·음성 데이터의 AI 학습 활용에 대한 동의를 처리한다 |
| **사전 조건** | 회원가입 또는 서비스 이용 중 |
| **사후 조건** | 동의 내역이 기록되고 이후 데이터 처리에 반영된다 |

---

## 기본 흐름

1. 시스템이 이용약관 및 개인정보처리방침을 제공한다
2. 영상·음성 데이터의 AI 학습 활용에 대한 별도 동의 항목을 표시한다
3. 사용자가 동의 여부를 선택한다
4. 시스템이 동의 내역을 기록한다

---

## 대안 흐름

- **3a.** 동의를 거부하더라도 서비스 기본 이용은 가능하며, 해당 데이터는 AI 학습에 활용되지 않는다

---

## 비고

- 동의 내역은 `user_consents` 테이블에 버전과 함께 저장된다
- 약관 버전이 변경될 경우 재동의를 유도한다
- 녹화 파일 메타데이터(`media_recordings`)에는 수집 시점의 동의 여부가 스냅샷으로 저장된다
