# 시퀀스 다이어그램 (Sequence Diagrams)

> 가상면접 웹 서비스의 주요 흐름별 시퀀스 다이어그램 목록

---

## 참여자 (Participants)

| 식별자 | 설명 |
|--------|------|
| `User` | 브라우저를 통해 서비스를 이용하는 사용자 |
| `Frontend` | React.js 클라이언트 |
| `Backend` | Django + DRF API 서버 |
| `DB` | PostgreSQL (관계형 DB) |
| `VectorDB` | 벡터 데이터베이스 |
| `LLM` | 대규모 언어 모델 (질문 생성, 파싱, 피드백) |
| `Storage` | 오브젝트 스토리지 (S3 등) |
| `Crawler` | 채용공고 크롤러 (비동기 워커) |
| `Scheduler` | 주기적 배치 작업 스케줄러 |
| `Email` | 이메일 발송 서비스 |
