# 가상면접 서비스 기획 문서

---

## 0. 서비스 한줄 요약

> AI 기반 가상 면접 연습 플랫폼으로, 매일 꾸준한 면접 연습 습관을 형성하고 개인화된 피드백으로 실전 면접 역량을 키운다.

---

## 1. 타깃 사용자

**1차 타겟: 신입 IT/SW 취업준비생**
- 코딩 테스트와 기술 면접을 동시에 준비하는 디지털 네이티브 구직자
- 회사별 맞춤 질문과 인성면접 STAR 기법 연습이 필요한 사용자

**2차 타겟: 비IT 대기업 지원자**
- AI역량검사·AI면접이 필수화된 대기업(삼성, LG, SK, 현대 등) 지원자
- 자소서 기반 예상 면접 질문 연습이 필요한 사용자

**3차 타겟: 이직 준비 경력직**
- 시간이 부족하여 효율적인 준비가 필요한 경력 3~10년차
- 시니어 레벨 시스템 디자인, 행동면접, 리더십 질문 연습이 필요한 사용자

**B2B 타겟**
- 대학 취업센터, 코딩 부트캠프, 정부 직업훈련 기관

---

## 2. Pain Points

- 한국 대기업의 AI 면접 도입이 보편화되었지만(2021년 기준 구직자 82% 경험), 전용 연습 플랫폼이 없다
- 기존 서비스(사람인 AI면접 등)는 세션 단위 과금(1회 10,000원)으로 꾸준한 연습이 어렵다
- 면접 준비가 일회성 이벤트에 그쳐 습관 형성이 되지 않는다
- 자소서 내용이 면접 질문으로 이어지는 한국 고유의 채용 흐름을 지원하는 서비스가 없다
- 연습 후 구체적이고 행동 가능한 피드백을 받기 어렵다
- 25~29세 취업자 수가 전년 대비 98,000명 감소(12년 만에 최악)하는 등 취업난이 심화되고 있다

---

## 3. 시장

**글로벌 시장**
- AI 면접/준비 도구 시장: 2025년 기준 약 $500M~$2B, 연 15% CAGR 성장
- HR 분야 AI 시장: 2024년 $7.01B → 2034년 $30.77B 예상
- Yoodli: 12개월간 900% 매출 성장, 기업가치 $300M+(6개월 만에 3배)
- Final Round AI: 2025년 1월 $6.88M 시드 라운드 (오버서브스크라이브)

**한국 시장**
- 연간 약 543,000명의 대졸자가 취업 시장 진입, 취업률 69.5%로 하락 추세
- 320개+ 기업이 AI 채용 도입 → 후보자측 AI 준비 도구 수요 급증
- 정부 AI 인재 양성 투자: 2026년 2,540억 원 배정(24만 명 교육)
- 한국 전용 독립 면접 연습 플랫폼은 사실상 공백 상태

**TAM/SAM/SOM (한국)**
- TAM: 150~1,800억 원 (연간 활성 구직자 500K~1M명 기준)
- SAM: 36~432억 원 (디지털 네이티브 구직자 중 AI 도구 관심층)
- SOM: 6~24억 원 (초기 1~2년, IT/SW 직군 + 대기업 지원자)

## 4. 주요 기능 및 시나리오

### 회원 관리
- 이메일/비밀번호 회원가입 및 로그인
- 이메일 인증 후 서비스 이용 가능
- 프로필 작성: 닉네임, 희망 직군, 현재 직업/직책
- 이력서, 채용공고, 자기소개서 업로드 및 관리

### 가상 면접
- **면접 방식 선택**
  - 꼬리질문 방식: 초기 질문 → 답변 기반 꼬리질문 1~3개 생성
  - 전체 프로세스 방식: 자기소개 → 지원동기 → 직무 질문 순서로 실전 면접 전체 흐름 연습
- **모드 선택**
  - 연습 모드: 버튼 클릭으로 답변 시작
  - 실전 모드: 5~30초 랜덤 대기 후 자동 시작
- 사용자 이력서·채용공고 기반 개인화 질문 생성 (LLM + 벡터 검색)
- 질문은 화면 표시 + TTS 음성으로 동시 제공
- 답변은 STT(크롬 브라우저 실시간 변환)로 텍스트 변환
- 침묵 감지(5초 이상, 특정 데시벨 이하) 및 맥락 기록
- 시선/고개 이탈 빈도 측정 (MediaPipe 기반)
- 면접 종료 후 리뷰 리포트 제공 (전반적 피드백, 침묵 분석, 시선 분석, 질문별 피드백)

### 스트릭 시스템
- 가상 면접 완료 시 스트릭 적립 (하루 1회)
- 전날 미참여 시 스트릭 초기화 (매일 자정 스케줄러)
- 현재 연속 일수, 역대 최장 기록, 참여 날짜 달력 시각화
- 마일스톤 달성 시 보상 자동 지급 (Pro 기능 사용 횟수 등)

### 요금제
- 무료 플랜: 기본 기능 제공, 일부 기능 횟수 제한
- Pro 플랜: 시선 추적, 실전 모드 등 고급 기능 무제한 이용

## 5. 주요 기술

| 영역 | 기술 |
|------|------|
| 프론트엔드 | React (TypeScript), STT (Web Speech API), MediaPipe (시선 추적) |
| 백엔드 | Django + Django REST Framework (Python) |
| LLM | 미정 |
| 데이터베이스 | PostgreSQL + pgvector (관계형 + 벡터 검색 통합) |
| 캐시 | ElastiCache Redis (세션, LLM 응답 캐싱) |
| 파일 저장 | S3 (이력서, 영상, 오디오) |
| 비동기 처리 | AWS SQS + Lambda |
| 스케줄링 | EventBridge Scheduler (스트릭 만료, 알림 발송) |
| TTS | 서버 생성 TTS 오디오 파일 제공 |
| 인증 | AWS Cognito (이메일 + 소셜 로그인 확장 가능) 고려 필요 |
| CDN | CloudFront |
| IaC | AWS CDK (TypeScript) |

## 7. 차별성

| 경쟁자 | 매일메일의 차별점 |
|--------|-----------------|
| 사람인 AI면접 | 세션 단위 과금이 아닌 월정액 구독. 매일 습관 형성에 특화된 독립 제품 |
| Final Round AI | 실시간 치팅 도구가 아닌 실력 향상 중심의 윤리적 포지셔닝 |
| Yoodli | 한국어 네이티브 + 한국 면접 문화(자소서→면접, 인성면접, 존댓말) 특화 |
| Google Warmup | 단순 패턴 인사이트가 아닌 상세 AI 평가 + 점수 + 개인화 피드백 |
| 제네시스랩 뷰인터HR | B2B 채용 솔루션이 아닌 B2C 자기 주도 연습 플랫폼 |

**핵심 차별화 3가지**
1. **습관 형성 메커니즘**: Duolingo식 스트릭 시스템으로 매일 면접 연습 루프 형성. 7일 스트릭 달성 사용자는 코스 완료율이 3.6배 높다는 데이터 기반 설계
2. **자소서→면접 파이프라인**: 한국 채용 고유의 흐름(자소서 내용 → 면접 질문)을 자동화. 해외 경쟁자가 쉽게 복제할 수 없는 문화적 깊이
3. **데이터 플라이휠**: 모든 사용자 답변이 AI 피드백 모델을 개선하고 질문 데이터베이스를 풍부하게 만드는 복리 경쟁 우위
