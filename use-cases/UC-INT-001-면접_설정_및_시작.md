# UC-INT-001 가상 면접 설정 및 시작

| 항목 | 내용 |
|------|------|
| **액터** | 일반 사용자, Pro 사용자 |
| **목표** | 가상 면접 세션을 설정하고 시작한다 |
| **사전 조건** | 로그인 상태, 프로필 작성 완료 |
| **사후 조건** | 면접 세션이 생성되고 첫 번째 질문이 제시된다 |

---

## 기본 흐름

1. 사용자가 지원 컨텍스트를 설정한다
   - 사전에 등록된 지원 현황(기업, 채용공고, 면접 단계) 목록에서 선택한다
   - 또는 이 단계에서 직접 입력한다
     - 지원 희망 기업명 (시스템 목록 선택 또는 자유 입력)
     - 채용공고 내용 (텍스트 직접 입력 또는 파일/URL 업로드)
     - 면접 단계 (예: 1차 면접, 2차 면접, 임원 면접 등)
     - 지원 직무 (시스템 목록 선택 또는 자유 입력)
2. 사용자가 면접 방식을 선택한다
   - 꼬리질문 방식: 하나의 주제에 대해 꼬리 질문을 이어가는 심층 면접
   - 전체 프로세스 방식: 자기소개부터 마무리까지 전 과정 연습
3. 사용자가 진행 모드를 선택한다
   - 연습 모드: 사용자가 준비 완료 버튼을 눌러 직접 답변을 시작한다. 결과가 스트릭에 반영된다.
   - 실전 모드: 질문 제시 후 5~30초 랜덤 대기 시간이 주어지고, 대기 후 자동으로 답변 녹화가 시작된다. 결과가 스트릭에 반영된다.
4. 시스템이 사용자 정보(직군, 직업, 지원 컨텍스트, 경향 분석, 업로드 문서)를 기반으로 면접 주제 n개를 LLM으로 생성한다
5. 시스템이 각 주제에 대해 벡터 DB에서 유사 자료를 검색한다 (RAG)
   - 사용자 업로드 문서 (`user_document_chunks`)
   - 선택된 채용공고 청크
   - 시스템 지식 베이스 (`system_knowledge_chunks`)
6. 시스템이 주제 및 유사 자료를 활용하여 면접 질문을 생성한다
7. 사용자가 카메라·마이크 권한을 허용하고 면접을 시작한다

---

## 대안 흐름

- **1a.** 사전 등록된 지원 현황이 없는 경우 직접 입력 폼을 바로 표시한다
- **1b.** 지원 컨텍스트 설정을 건너뛸 수 있으며, 이 경우 사용자 프로필(직군, 직업)만으로 면접 주제를 생성한다
- **2a.** 요금제에 따라 선택 불가한 방식/모드가 있는 경우 업그레이드를 안내한다
- **7a.** 카메라 또는 마이크 권한이 거부된 경우 안내 메시지를 표시한다
