# UC-GMF-002 스트릭 현황 조회

| 항목 | 내용 |
|------|------|
| **액터** | 일반 사용자, Pro 사용자 |
| **목표** | 현재 연속 참여 일수와 과거 참여 날짜 기록을 확인한다 |
| **사전 조건** | 로그인 상태 |
| **사후 조건** | 스트릭 현황 화면이 표시된다 |

---

## 기본 흐름

1. 사용자가 스트릭 현황 페이지에 접근한다
2. 시스템이 다음 정보를 조회하여 표시한다
   - 현재 연속 참여 일수 (`streaks.current_streak`)
   - 역대 최장 연속 참여 일수 (`streaks.longest_streak`)
   - 가상 면접에 참여한 날짜 목록 (`streak_participation_logs`에서 조회, 달력 형태로 시각화)
   - 다음 보상까지 남은 스트릭 일수 (`streak_reward_policy`에서 다음 마일스톤 계산)

---

## 비고

- 달력 뷰에서 면접 참여 날짜는 별도 표시(예: 색상 강조)로 구분된다
- 관련 테이블: `streaks`
- 보상 정보 확인: [UC-GMF-004-스트릭_보상_수령.md](./UC-GMF-004-스트릭_보상_수령.md)
- 참여 날짜 이력 출처: `streak_participation_logs` (스트릭 적립 시 자동 기록)
