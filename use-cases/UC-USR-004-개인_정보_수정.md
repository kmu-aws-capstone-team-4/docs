# UC-USR-004 개인정보 수정

| 항목 | 내용 |
|------|------|
| **액터** | 일반 사용자, Pro 사용자 |
| **목표** | 프로필 정보를 수정한다 |
| **사전 조건** | 로그인 상태 |
| **사후 조건** | 변경 사항이 저장되고 이후 면접 질문 생성에 즉시 반영된다 |

---

## 기본 흐름

1. 사용자가 개인정보 수정 페이지에 접근한다
2. 사용자가 수정할 항목을 변경한다
   - 비밀번호
   - 닉네임
   - 희망 직군
   - 직업
3. 시스템이 변경 사항을 저장한다
4. 직군/직업 정보가 변경된 경우, 이후 면접 질문 생성에 즉시 반영된다

---

## 비고

- 희망 직군과 직업은 면접 질문 생성 시 그때그때 조회되는 동적 정보이므로, 수정 즉시 다음 면접 세션부터 반영된다
- 진행 중인 면접 세션에는 영향을 주지 않는다 (세션 시작 시점의 스냅샷 사용)
