# UC-INT-003 전체 프로세스 방식 면접 진행

| 항목 | 내용 |
|------|------|
| **액터** | 일반 사용자, Pro 사용자 |
| **목표** | 실제 면접 전 과정을 처음부터 끝까지 연습한다 |
| **사전 조건** | 면접 세션 시작 완료 ([UC-INT-001](./UC-INT-001-면접설정및시작.md)), 전체 프로세스 방식 선택 |
| **사후 조건** | 모든 질문이 완료되고 면접 세션이 종료된다 |

---

## 기본 흐름

1. 가상 면접관이 자기소개 질문으로 면접을 시작한다
2. 사용자가 답변한다 (연습/실전 모드에 따라 시작 방식 상이)
3. 시스템이 침묵 및 시선 이탈을 감지하고 기록한다
4. 지원동기 질문으로 이어진다
5. 사용자가 답변한다
6. 기타 면접 질문들이 순서대로 제시된다
7. 모든 질문 완료 후 면접 세션이 종료되고 리포트 생성으로 이어진다

---

## 대안 흐름

- **2a.** 사용자가 답변 도중 세션을 중단하면 `abandoned` 상태로 저장된다

---

## 비고

- 꼬리 질문은 제공되지 않는다
- 질문 순서: 자기소개 → 지원동기 → 기타 질문
